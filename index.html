Projet OpenQuizz

Objectif : créer une application Swift

Cette application est un jeu qui permet à l'utilisateur de répondre à un quizz de 10 questions. Il peut choisir le thème des questions abordées grâce à un UISegmentedControl lié à un API. Il peut aussi recommencer une partie quand il le désire en cliquant simplement sur le bouton New Game. Les éléments ont des "constraints" pour que l'application fonctionne sur tous les appareils (mobiles et tablettes IOS).
Ce jeu utilise des objets UIKit, des animations, le modèle MVC, des appels réseaux et des tests unitaires.

  • Technologie utilisée : Swift et API
  • Session été 2020
  • Professeur : Vincent Leduc
